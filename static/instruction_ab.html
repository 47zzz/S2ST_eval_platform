Expressive Speech-to-Speech Translation 評測介面 Expressive Speech-to-Speech Translation Evaluation
第二部分 — 偏好評測 (A/B Test) Part 2 of 2 — Pairwise Preference Evaluation (A/B Test)

任務說明 Task Overview

本評測旨在比較不同語音生成模型在情緒表達上之表現。受測者將聆聽一段基準語音（Source Audio），以及由兩組不同模型（Model A 與 Model B）所生成之翻譯語音。

請針對上述兩段模型語音進行對比，並完成以下兩項評估指標：

This evaluation aims to compare the emotional expression performance of different speech generation models. You will listen to a source audio (Source Audio) and translated speech generated by two different models (Model A and Model B).

Please compare the two model outputs and complete the following two evaluation metrics:

一、情緒表徵相近度 (Emotion Similarity Comparison) 1. Emotion Similarity Comparison

請以基準語音為參照，在排除語者特徵（如語者身份、性別、年齡、音色差異等）之情況下，比較 Model A 與 Model B，判定何者之「情緒表徵」與基準語音更為契合。

評定選項：

  • Model A 更為相近
  • Model B 更為相近
  • 平手 / 相似 (Tie / Similar)

Using the source audio as reference, and excluding speaker characteristics (such as speaker identity, gender, age, timbre differences, etc.), compare Model A and Model B to determine which exhibits emotional expression more closely matching the source audio.

Options:

  • Model A is more similar
  • Model B is more similar
  • Tie / Similar

「平手 / 相似 (Tie / Similar)」選填規範：

請審慎聆聽兩段音訊並盡可能區辨其差異。唯有在您的聽覺感知閾值內，確切無法分辨兩者情緒表徵之優劣時，方可選擇「Tie / Similar」。若能感知任何微小之勝負傾向，請務必選擇表現較佳之選項，避免過度使用平手選項。

"Tie / Similar" Selection Guideline:

Please listen carefully to both audio samples and try to differentiate them as much as possible. You may only select "Tie / Similar" when you truly cannot distinguish any difference in emotional expression within your perceptual threshold. If you perceive even the slightest advantage, please select the better option. Avoid overusing the Tie option.

Presence of Speech-embedded Nonverbal Cues 評定標準 Presence of Speech-embedded Nonverbal Cues

若音檔中沒有非語言聲音，則無須勾選任何選項。只有在觀察到下列情形時，才選擇對應的選項。

If no nonverbal cues are present, leave the selection blank. Only select an option when you observe the following.

  • Laughing - 邊笑邊講話 (Laughing while speaking) Laughing while speaking

    操作性定義：說話者在輸出具語意內容之語音時，聲音中伴隨笑聲，或因笑而引發之氣音與聲帶顫抖。

    Definition: While producing meaningful speech, the voice clearly carries laughter, or breath/vocal trembling caused by laughter.

  • Crying - 邊哭邊講話 (Crying while speaking) Crying while speaking

    操作性定義：說話者在輸出具語意內容之語音時，聲音中伴隨哭腔、啜泣（Sobbing）、哽咽，或因哭泣導致之呼吸急促與發音顫抖。

    Definition: While producing meaningful speech, the voice clearly includes crying, sobbing, a choked voice, or accelerated breathing/vocal trembling caused by crying.

核心原則：言語重疊性 (Speech Overlap)

非語言特徵須在「語言表達（字詞或語句）」期間發生：

  • 不予採計：僅有純粹笑/哭聲（如單獨大笑、大哭），未伴隨語意內容。
  • 必須勾選：特徵呈現於語句之中，不論特徵強弱或持續時間長短。

Core Principle: Speech Overlap

Nonverbal features must occur during meaningful speech (words or sentences):

  • Exclude: Pure laughter/crying only (e.g. standalone laughter, sobbing) with no verbal content.
  • Must select: Feature is present within an utterance, regardless of intensity or duration.

The following three anchor examples demonstrate the correct judgements for reference.

評測流程與注意事項 Evaluation Procedure & Notes

  • 為防範順序效應（Order effect）之偏誤，各組 Model A 與 Model B 之實際對應模型皆採隨機指派。
  • 請確保已完整評估「情緒表徵相近度」與「非語言資訊」兩項指標後，方可點擊 "Submit & Next" 進入次一評測組別。
  • To prevent order effects, the actual models assigned to Model A and Model B are randomly assigned for each set.
  • Please ensure you have completed both the "Emotion Similarity Comparison" and "Nonverbal Cues" assessments before pressing "Submit & Next" to proceed to the next set.
